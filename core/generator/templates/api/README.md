A glowing-fishstick API generated by `fishstick-create`.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:{{port}}`.

## Routes

Core routes (provided by `@glowing-fishstick/api`):

- `GET /healthz` — health check
- `GET /readyz` — readiness check
- `GET /livez` — liveness check
- `GET /metrics/memory` — memory metrics
- `GET /metrics/runtime` — runtime metrics

API route (from your plugin):

- `GET /api/my-feature` — example API endpoint

## Structure

```text
{{projectName}}/
  package.json
  src/
    server.js        — thin entrypoint (compose + boot)
    api.js           — your plugin (routes + lifecycle hooks)
    config/env.js    — config overrides
    routes/router.js — example route
```

## Adding Features

Edit `src/api.js` to add routes and lifecycle hooks. Follow the plugin contract:

```js
export function myPlugin(app, config) {
  app.use(myRoutes());
}
```

See the [glowing-fishstick docs](https://github.com/jeffcaradona/glowing-fishstick) for full documentation.
