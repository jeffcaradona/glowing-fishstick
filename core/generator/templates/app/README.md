# {{projectName}}

A glowing-fishstick application generated by `fishstick-create`.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

Core routes (provided by `@glowing-fishstick/app`):

- `GET /` — landing page
- `GET /healthz` — health check
- `GET /readyz` — readiness check
- `GET /livez` — liveness check
- `GET /admin` — admin dashboard
- `GET /admin/config` — config viewer

App route (from your plugin):

- `GET /my-feature` — example feature page

## Structure

```text
{{projectName}}/
  package.json
  src/
    server.js        — thin entrypoint (compose + boot)
    app.js           — your plugin (routes + lifecycle hooks)
    config/env.js    — config overrides
    routes/router.js — example route
    views/           — Eta templates
    public/          — static assets (css/, js/)
```

## Adding Features

Edit `src/app.js` to add routes, lifecycle hooks, and nav links. Follow the plugin contract:

```js
export function myPlugin(app, config) {
  app.locals.navLinks.push({ label: 'New', url: '/new' });
  app.use(myRoutes(config));
}
```

See the [glowing-fishstick docs](https://github.com/jeffcaradona/glowing-fishstick) for full documentation.
